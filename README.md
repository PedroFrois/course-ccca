Para iniciar o projeto:

`npm install typescript jest @types/jest ts-node ts-jest`
`npx tsc --init`

Tenha o postgres rodando. Crie um banco. 
Ex: `createdb app`
Crie o esquema `psql -d app -f create.sql`
